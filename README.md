About
I'm a full-stack developer who builds every layer of a product single-handedly, from the first idea to a live, working application. My home base is the JavaScript/TypeScript ecosystem, but I'm comfortable switching to Python or PHP when a project calls for it.
What really sets me apart isn't just being able to write code. It's being able to take a product from an idea all the way into the hands of real users, on my own. That goes beyond frontend and backend: Linux server administration, self-hosted deployments, Cloudflare configuration, Docker, reverse proxy setup and App Store / Google Play release workflows are all part of how I ship.
I actively use AI-assisted development and well-designed prompt strategies to compress work that would traditionally take weeks into just a few days. But the decisions are still mine: architecture, stack choices, product direction and infrastructure all sit on my shoulders.
